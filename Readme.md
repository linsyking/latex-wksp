# $\LaTeX$ 2023 Summer Workshop 

**A Practical Introduction to the $\LaTeX$ typesetting system**.

Run `make` to compile the final slide.

## Topics

- General Introduction to LaTeX
- Overleaf/VS Code Setup
- Text Rendering
- Math Rendering
- Commonly-used Environments
- Templates
- Tikz
- Beamer
- BibTeX
